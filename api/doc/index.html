get /api/v1/events/{month}/{day}
Get events for a specific date (apiV1EventsMonthDayGet)
Returns historical events that occurred on a given month and day.

Path parameters

month (required)
Path Parameter — Month (1-12)
day (required)
Path Parameter — Day of the month (1-31)

Query parameters

issue_id (optional)
Query Parameter — Retrieve a specific event by its unique numerical issue ID.
year (optional)
Query Parameter — Filter events by a specific year.
page (optional)
Query Parameter — Page number for pagination. default: 1

Return type

inline_response_200

Example data

Content-Type: application/json
{
  "date" : "31 January",
  "events" : [ {
    "page_number" : 2,
    "issue_id" : 102345,
    "year" : 1944,
    "description" : "Skeleton Order of Battle, 15 Indian Corps, 1st January 1944",
    "title" : "Tere."
  }, {
    "page_number" : 2,
    "issue_id" : 102345,
    "year" : 1944,
    "description" : "Skeleton Order of Battle, 15 Indian Corps, 1st January 1944",
    "title" : "Tere."
  } ]
}
